
Setuptools
===========

wget https://pypi.python.org/packages/source/s/setuptools/setuptools-1.0.tar.gz
tar xf setuptools-1.0.tar.gz
cd setuptools-1.0
python setup.py install

Redis
=======

Installation
Download, extract and compile Redis with:

wget http://download.redis.io/releases/redis-2.6.15.tar.gz
tar xf redis-2.6.15.tar.gz
cd redis-2.6.15
make
